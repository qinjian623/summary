**⭕ 一句话总结**  
心理防御机制是人在面对压力、焦虑或冲突时，为维持心理平衡而产生的无意识反应；它们可分为由不成熟到成熟的不同类型，使用的方式及频率可反映心智成熟度。  

---

**📋 防御机制分类与说明（由不成熟到成熟）**

| 成熟度 | 机制名称 | 英文 | 核心特点 | 举例 |
|--------|----------|------|-----------|------|
| **原始/不成熟** | 1. 否认 | Denial | 拒绝承认或接受现实，扭曲事实 | 小明宠物车祸去世，却坚称只是失踪 |
|        | 2. 退化 | Regression | 遇挫时退回到幼稚行为模式，依赖他人 | 孩子或成年人吵闹撒泼 |
|        | 3. 行动化 | Acting Out | 直接用极端行为释放情绪 | 砸东西、自残、吸毒 |
|        | 4. 解离 | Dissociation | 暂时失去记忆/人格完整性，逃避崩溃 | 车祸后失忆，不认识重要人物 |
|        | 5. 投射 | Projection | 将无法接受的冲动/想法归咎他人 | 作弊后称别人也作弊 |
|        | 6. 反向作用 | Reaction Formation | 做与真实想法相反的行为来隐藏感受 | 喜欢某人却表现为讨厌或攻击 |

| **较成熟** | 7. 压抑 | Repression / Suppression | 将痛苦记忆/情感压入潜意识或暂时隐藏 | 经历空难后刻意不回想细节 |
|            | 8. 转移 | Displacement | 将情绪发泄在代替对象上 | 被老板骂→回家对家人发火 |
|            | 9. 合理化 | Rationalization | 用“合理”理由解释不良行为或失败 | 输球后称自己是保存体力 |

| **成熟** | 10. 幽默 | Humor | 用玩笑化解尴尬或困境 | 苏格拉底被妻子泼水后幽默回应 |
|          | 11. 补偿 | Compensation | 在另一领域取得成就，弥补弱项 | 学业差→音楽上努力成功 |
|          | 12. 升华 | Sublimation | 将冲动转化为社会可接受的形式 | 喜欢玩火→成为火舞表演者 |

---

**📌 关键观点**  
- 防御机制由弗洛伊德提出，安娜·弗洛伊德及其他心理学家完善。  
- “不成熟”机制更接近童年行为模式；使用频率过高可能暗示心理不成熟甚至病态。  
- 健康人会在不同情境中使用多种机制，并非完全避免原始机制。  
- 成熟的防御机制通常更具建设性，能积极帮助个人适应环境与缓解心理压力。  

---

**💡 总结提炼**  
人的心智成熟度可部分从其防御机制的类型与使用习惯看出：  
- **不成熟机制** 常以逃避现实、情绪宣泄为主，短效但易造成负面影响；  
- **成熟机制** 能在解决或缓和压力的同时带来社会认可与个人成长。
