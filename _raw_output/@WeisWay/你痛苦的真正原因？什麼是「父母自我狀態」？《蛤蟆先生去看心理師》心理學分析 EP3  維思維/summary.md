**内容总结：**

本集节目延续《蛤蟆先生去看心理师》第三集的情节，围绕蛤蟆与心理师仓路的第五次会面，探讨 **抑郁症**的误解、**心理游戏** 以及 **父母自我状态** 对个人的影响。

---

### 一、故事情节概览

1. **老欢来访冲突**
   - 蛤蟆情绪有所改善（情绪温度计从 1-2 提升到 5-6），但仍在恢复中。
   - 老欢带着“批评+施压”的态度，要求蛤蟆辞去学校董事职位，并由自己接替。
   - 蛤蟆在强硬气场下陷入无力与自我怀疑，犹豫是否辞职。

2. **心理师仓路分析**
   - 指出蛤蟆陷入心理游戏 **PLOM（Poor Little Old Me，可怜的小可怜）**：通过贬低自己、退出职位，避免与老欢正面冲突，却在“退让”中获得安全感。
   - 共谋机制：蛤蟆与老欢表面对立，实则在无意识中配合，使自己成为受害者。
   - 解释抑郁症与“普通低落”的区别——抑郁症是长期、持续的精神疾病，需要专业帮助，不是“看开点”就能解决。

3. **父母自我状态**
   - 老欢表现出“挑剔型父母”状态，严厉、道德化、批评性强，激发蛤蟆童年被父亲威压的记忆，使其习惯性退缩。
   - 仓路通过图示解释“父母之我状态”是从父母那里内化的价值观与规则。即使蛤蟆不对他人挑剔，也可能将这种状态对准自己（内在批评者）。
   - 蛤蟆承认自己常常“审判自己”，陷入自我批判和严苛的自我惩罚。

4. **自我接纳的重要性**
   - 仓路强调：蛤蟆的核心问题不在于外部支持的缺乏，而在于缺乏对自己的支持。
   - 改变的关键是把内在批判者转化成支持者，允许不完美、给自己善意与保护。
   - 自我接纳的例子：
     - 从“我是没用的，我该辞职”转向
     - “我现在状态不好，需要休息，但我依然有价值，恢复后可以继续努力”。

5. **结果**
   - 后续蛤蟆给自己的情绪温度计打了 8分，并在面对老欢时找到了更好的回应方式。
   - 情绪与自我认同明显改善。

### 二、核心概念整理

| 概念 | 解释 | 本案例体现 |
| ---- | ---- | -------- |
| **抑郁症** | 持续性低落、兴趣丧失、失眠、体重变化、无价值感等，需专业治疗，不是简单的情绪低潮 | 蛤蟆的长期低落与自我否定 |
| **心理游戏 PLOM** | 在互动中通过成为受害者或无助者来回避更直接的冲突或压力，同时获得心理安全感 | 蛤蟆以“辞职对大家好”回避与老欢对抗 |
| **共谋** | 无意识配合对方，制造让自己痛苦的情境 | 蛤蟆的退让与老欢的批判形成隐性配合 |
| **父母自我状态** | 从父母那里内化的价值观与行为模式，可表现为挑剔或保护两面 | 老欢表现挑剔型父母；蛤蟆将批评对准自己 |
| **内在批评者** | 父母之我状态内化后，对自己的苛责评价 | 蛤蟆承认会“审判自己”并惩罚自己 |
| **自我接纳** | 将内在批评者转化为支持者，对自己友善，允许不完美 | 改变后，蛤蟆能在休息的同时保持自我价值感 |

---

**一句话总结：**  
蛤蟆的痛苦根源不仅在于抑郁症，更在于长期内化的严厉“父母之我状态”对自我的批判，他通过心理游戏回避冲突但失去自我价值；真正的改善来自拆解内在批评者，学会自我接纳与善待自己。
